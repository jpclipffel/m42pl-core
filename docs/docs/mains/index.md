# Mains

The M42PL core package comes with different _mains_, aka. entry points
to runs your pipelines.

* [repl]: Runs M42PL as a REPL (interactive command interpreter)
* [run]: Runs a M42PL script
* [grammar]: Prints M42PL EBNF grammar
* [parse]: Parse a M42PL script
