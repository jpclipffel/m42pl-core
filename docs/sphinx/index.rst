Welcome to M42PL technical documentation!
=========================================

M42PL is a both a data flow / data processing language and an implementation
of this language, written in Python.

This technical documentation describes the internal components of the
language implementation, and should help interested developpers to extends /
improve / correct / ... the project.

The documentation is split in two sections:

* Overview: Introduces the internal components and explains how to extends them
* API: Code documentation generated from the docstrings
