# Components

M42PL is built on 5 kind of components, each of them providing a set of core
functionnalities and each of them being extendable and replacable.

* [Core](./core.md): The core project
* [Commands](./commands.md): The language commands and constructs
* [KVStores](./kvstores.md): The key-value interfaces
* [Encoders](./encoders.md): The data encoding/decoding codecs
* [Dispatchers](./dispatchers): The pipelines runners/executors/schedulers/...
